
#  th1. 길이와 시간

## -1 ) 자연 세계의 길이와 시간 규모

**(1) 자연 세계:** 자연 세계는 크게 두 부류로 나눌 수 있는데,
미시 세계와 거시 세계로 구분 할 수 있다.

| 구분    | 미시 세계                    | 거시 세계                      |
| ----- | ------------------------ | -------------------------- |
| 정의    | 원자 수준의 아주 작은 물체나 현상들의 규모 | 미시 세계보다 큰 물체나 현상들의 규모      |
| 탐구 대상 | 원자핵, 원자, 분자, 이온 ...etc   | 동물, 암석, 바다, 행성, 태양계 ...etc |
| 관측 장비 | 전자 현미경, 입자 가속기 ...etc    | 우주 망원경, 천체 망원경 ...etc      |

**(2) 규모:** 과학에서는 넓은 범위의 자연 현상을 어느 정도의 범위로 구분 지어 표현하는데, 이러한 공간과 시간의 크기 범위를 나타내는 것을 규모라고 한다

다시 쉽게 풀어보면, 10명 중 9명이 볼 때 "크다.." 싶으면 규모라고 봐도 좋다.
Ex) 
2인 예약 : 규모가 큰 예약 ? -> (x)
40인 예약 : 규모가 큰 예약 ? -> (O)

## -2) 길이와 시간 측정

**(1) 길이 측정 방법의 발전:** 
- 과거 : 몸의 일부분(팔,다리...etc) , 일정한 길이의 막대,기준이 되는 길이가 눈금으로 표시되어 있는 자 
- 현대 : 빛을 이용 or GPS or 전자 현미경 ...etc

**(2) 시간 측정 방법의 발전:**
- 과거 : 지구의 자전 or 공전을 이용해서 시간을 측정
- 현대 : 원자에서 방출 or 흡수 되는 빛을 이용하여 측정

## -2) 기본량과 유도량

**(1) 기본량:** 길이나 시간처럼 다른 물리량으로 정의 할 수 **없는** 가장 기본이 되는 7개의 물리량을 의미한다
- 종류 : 길이, 질량, 시간, 온도, 광도, 전류, 물리량

| 기본량 | 기본 단위     |
| --- | --------- |
| 길이  | m (미터)    |
| 질량  | kg (킬로그램) |
| 시간  | s (초)     |
| 온도  | K (켈빈)    |
| 광도  | cd (칸델라)  |
| 전류  | A (암페어)   |
| 물질량 | mol (몰)   |

**유도량:** 기본량을 조합해 유도하는 물리량을 의미한다
Ex) 이동거리 / 시간 = 속력
